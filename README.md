# Calculadora-de-Encargos-e-Impostos
Este é um projeto de Calculadora de Encargos e Impostos que simula e compara a remuneração líquida nos regimes de trabalho CLT e PJ.

O Projero simula e compara a remuneração líquida nos regimes de trabalho CLT (considerando INSS e IRRF) e PJ (calculando impostos simplificados como Simples Nacional e Lucro Presumido). Ele fornece relatórios detalhados para auxiliar na decisão entre os modelos de contratação.
